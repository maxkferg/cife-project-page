Results

The proposed system is evaluated qualitatively and quantitatively. One of the core components is an novel 3D object detection method, namely Object R-CNN. Below, we compare the output of Object R-CNN the state-of-the-art Mask R-CNN architecture.

Comparison of worksite object detection using Mask R-CNN and Object R-CNN (ours). Both systems are trained on the same dataset, but our system predicts 3D position and size.

Comparison of worksite object detection using Mask R-CNN and Object R-CNN (ours).

Comparison of worksite object detection using Mask R-CNN and Object R-CNN (ours).

Our object detection network (Object R-CNN) reaches a similar accuracy as Mask R-CNN in 2D instance segmentation, but also predicts the size and location of objects in 3D space. However, errors in object detection and mobile robot localization can still lead to problems when updating the building information model. Our tracking system minimizes the impact of localization error and false-detections:

Comparison of the updated building information model without tracking system (left) and with tracking system (right).
