Dataset

A dataset was collected using the purpose-built mobile robot. The dataset includes 6121 RGB-D labelled objects across 1007 images. Objects are labelled with 2D segmentation masks and 3D bounding boxes.

The dataset is split into test and train sequences. The training sequence contains 5121 objects labelled across 821 RGB-D images. The test set consists of a further 1010 objects annotated across 186 images.
